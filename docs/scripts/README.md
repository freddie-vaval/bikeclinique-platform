## 📹 How to Create Each Video

### Step 1: Get Stock Footage
Open CapCut → Stock Media → Search keywords from each script

### Step 2: Add Text Overlays
Use bold text matching the script

### Step 3: Add Music
CapCut → Music → Search "upbeat" or "cinematic"

### Step 4: Export & Post
- TikTok: Direct from CapCut
- YouTube Shorts: Export → Upload
